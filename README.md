# 📡 Transfer Texto

Transferencia de texto bidireccional entre **Linux Mint** y **Termux** (Android) en la misma red WiFi.

[![Versión](https://img.shields.io/badge/versión-1.0.0-blue.svg)](https://github.com/TU_USUARIO/transfer-texto)
[![Licencia](https://img.shields.io/badge/licencia-MIT-green.svg)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.6+-yellow.svg)](https://www.python.org/)

## ✨ Características

- 🔄 Transferencia bidireccional de texto
- 📱 Compatible con Android (Termux) y Linux
- 🌐 Funciona en la misma red WiFi
- 👥 Soporte para múltiples clientes
- 📋 Historial de mensajes
- 🔒 Sin dependencias externas
- 🚀 Fácil instalación con un comando

## 📦 Instalación Rápida

### En Linux Mint (Servidor)
```bash
curl -sL https://raw.githubusercontent.com/urzagaste222/transfer-texto/main/scripts/instalar_linux.sh | bash

### En Termux (Cliente)
```bash
curl -sL https://raw.githubusercontent.com/urzagaste222/transfer-texto/main/scripts/instalar_termux.sh | bash

## Uso básico

- En Linux Mint
```bash
transfer-server

- En Termux
```bash
transfer-client
